Information

Various information about articles has been stored in dataset files. This information can include the name of article, type of article, related section, abstract, keywords, publisher, link, language and information of authors. This information is saved for all articles and existing conferences.

This dataset is presented in three formats: txt, json and xml. To get acquainted with the structure of dataset files, you can view an image of the tree structure of information, JSON Schema and DTD and XML Schema from the following links:

  • Tree structure
  • JSON Schema
  • DTD & XML Schema

The RTLTDS data warehouse contains article, conference, author, publisher and language dimension tables and the publication fact table. The schema of this data warehouse is visible at the following link:

  • Data warehouse Schema

For more information about the dataset, data warehouse and how to build them, you can see the related article of dataset from the following link.
