اطلاعات

اطلاعات مختلفی در ارتباط با مقالات در فایل‌های مجموعه داده ثبت شده اند. از جمله این اطلاعات میتوان به نام مقاله، نوع مقاله، بخش مربوطه، چکیده، کلمات کلیدی، منتشرکننده، لینک، زبان و اطلاعات مولفان اشاره کرد. این اطلاعات برای تمامی مقاله‌ها و تمامی کنفرانس‌های موجود ثبت شده اند. این مجموعه داده با سه فرمت json ،txt و xml ارائه شده است. برای آشنایی با ساختار فایل‌های مجموعه داده میتوانید تصویری از ساختار درختی اطلاعات، JSON Schema و DTD و XML Schema را از لینک‌های زیر مشاهده نمایید:

  • ساختار درختی
  • JSON Schema
  • DTD & XML Schema

پایگاه داده تحلیلی RTLTDS شامل جداول بعد article، conference، author، publisher و language و جدول حقیقت publish می‌باشد. طرح این پایگاه داده تحلیلی از طریق لینک زیر قابل مشاهده می‌باشد:

  • Data warehouse Schema

برای اطلاعات بیشتر در ارتباط با مجموعه داده، پایگاه داده تحلیلی و نحوه ساخت آنها میتوانید از طریق لینک‌های زیر مقالات مربوط به آنها را مشاهده بفرمایید.

Efficient process for preparing right-to-left text-based datasets for data mining and big data applications
Efficient approach for creating a data warehouse from a data repository of JSON documents
